# Security

Report vulnerabilities privately through GitHub Security Advisories for this repository or contact `dev@hol.org`.

Do not include secrets, tokens, private keys, `.env` contents, customer data, or local receipt payloads in public issues.

This plugin is a Codex instruction and helper package. The runtime security implementation lives in [`hashgraph-online/hol-guard`](https://github.com/hashgraph-online/hol-guard) and the published `hol-guard` package.
