# Product Style Map

## 用途

- 根据产品类型快速确定视觉语气、信息密度和交互节奏，避免在实现中来回摇摆。

## 推荐映射

| 产品类型 | 视觉方向 | 信息密度 | 交互节奏 |
|----------|----------|----------|----------|
| 企业后台 / 管理台 | 克制、清晰、结构优先 | 中高 | 稳定、反馈明确 |
| 数据看板 / 运营面板 | 信息优先、层级强 | 高 | 快速扫描、重点突出 |
| SaaS 工作台 | 可信、效率导向 | 中 | 连续任务流、低干扰 |
| 内容型站点 | 内容优先、留白明确 | 中低 | 阅读流畅、导航直觉 |
| 品牌展示 / 活动页 | 视觉氛围更强 | 低中 | 节奏感明显，但不能牺牲可读性 |

## 使用规则

- 一次需求只选一个主产品类型，必要时补一个次类型说明混合原因。
- 视觉方向和信息密度一旦选定，后续 token、布局、组件风格都要服从它。
