## Self-check

Before shipping any web generation:

1. Does this fit alongside refs from the matching direction?
2. Is the type distinctive (not Inter / Space Grotesk)?
3. Are there 2-4 signature motion moments?
4. Does the layout break "hero + 3-column features + CTA"?
5. Dark mode has warmth/coolness, not inversion?
6. Did I avoid every item in `web-bans.md`?
7. Did I avoid every phrase in `web-copy-voice.md` ban list?

If any "no" — restart that choice, don't ship near-misses.
