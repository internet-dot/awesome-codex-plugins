## 🚀 Usage

### Load my persona in Claude Code

```bash
ln -s $(pwd)/me ~/.claude/skills/vibe-portrait-personas/me
```

Then say: **"think like me"** or **"像我一样思考"** in any conversation.

### Sync across machines

```
/vibe-portrait
# → Choose "Sync with my portrait repo" when prompted
```

### Load community personas

```bash
ln -s $(pwd)/community ~/.claude/skills/vibe-portrait-personas/community
```
